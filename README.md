## Отчёт по анализу рынка недвижимости Санкт‑Петербурга и Ленинградской области
В рамках проекта выполнен комплексный анализ данных для определения перспективных сегментов рынка и сезонных тенденций с целью поддержки стратегии выхода агентства недвижимости на рынок двух регионов.

С помощью SQL‑запросов проведена агрегация и анализ данных по следующим направлениям:

Сегментация рынка. Данные сгруппированы по ключевым параметрам: локация (СПб и города ЛО), тип жилья, площадь и др. Привлекательность сегментов оценена через метрики активности объявлений — количество публикаций и среднее время нахождения в активном статусе. Выполнено кросс‑региональное сравнение для выявления наиболее перспективных сегментов.

Анализ сезонности. Данные агрегированы по временным периодам (месяцы и кварталы) с геопривязкой. Выявлены устойчивые сезонные паттерны динамики активности за несколько лет: определены пиковые и «низкие» сезоны, оценена амплитуда колебаний спроса и предложения.

- На базе полученных датасетов создан дашборд в DataLens:

- добавлены визуализации по сегментации рынка и сезонным трендам;

- внедрены интерактивные фильтры по локации, типу жилья и периоду;

- актуализированы KPI и ключевые метрики;

- обеспечена наглядность и удобство интерпретации данных при сохранении единого стиля дашборда.

Итоговый результат: сформирован аналитический фундамент для бизнес‑стратегии агентства. Определены приоритетные сегменты рынка недвижимости и оптимальные периоды для запуска маркетинговых активностей и публикации объявлений.
